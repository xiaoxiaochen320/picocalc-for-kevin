# PicoCalc for Kevin

A pocket-sized rabbit hole.

Because normal gifts are boring.

This felt much more *you*:

- tiny keyboard
- retro terminal vibes
- infinite opportunities to overengineer something unnecessary

---

## What is this?

PicoCalc is somewhere between:

- a calculator
- a pocket computer
- a retro terminal
- a programmable nerd toy

Basically:

**a tiny machine for coding, hacking, gaming, and making questionable technical decisions.**
